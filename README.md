# ✅ Todo App

A simple and responsive Todo App built using **HTML, CSS, and JavaScript**. This application allows users to add, update, and delete tasks dynamically. It demonstrates DOM manipulation, event handling, and basic CRUD operations on the client side without using any backend.

## 🚀 Features

- ➕ Add new tasks
- ✏️ Update existing tasks
- 🗑️ Delete tasks
- 📋 Display tasks instantly
- ⚡ Fast and lightweight
- 📱 Responsive design
- 🎨 Clean and user-friendly interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📖 How It Works

1. Enter a task in the input field.
2. Click the **Add Task** button.
3. The task is displayed in the list.
4. Click **Update** to edit the task.
5. Click **Delete** to remove the task.
